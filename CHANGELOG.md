## v1.4 (2026-03-07)

### 추가
- 시뮬레이션 모드 — 영상 없이 가상 자세 생성 및 인체공학 평가
- 3D 뷰 회전 (방향패드 + 마우스 드래그) 및 z축 이동
- 이미지/동영상 회전 및 반전 기능 (보기 메뉴 연동)
- 다중 인원 감지 시 가장 큰(가까운) 사람 자동 선택
- 포즈 감지 모델 선택 기능 (Lite/Full/Heavy, 등록 전용)
- 모델 미존재 시 자동 다운로드 다이얼로그 표시
- 스켈레톤 영역 클릭 시 편집 모드 자동 진입
- 동영상 로드 시 첫 프레임 자동 인체 감지
- 모든 랜드마크 visibility 최소 0.5 강제 적용
- 프로젝트 저장/로드 시 변환 설정(회전/반전) 연동

### 수정
- Skeleton Analyzer → IMAS 리네이밍
- REBA, OWAS 분석을 유료 기능으로 전환 및 잠금 화면 추가
- QMessageBox를 커스텀 다크 테마 다이얼로그로 전면 교체
- 캡처 저장 경로에 captures/ 하위 폴더 자동 적용
- 시뮬레이션 모드 진입/해제 시 기존 스켈레톤 백업/복원
- 앱 종료 시 PoseDetector GC bus error 방지
- capture_save 빈 문자열 시 프로젝트 루트 삭제 방지

### 문서
- 도움말에 5개 안전지표 상세 계산 공식 및 조회 테이블 추가
- 시뮬레이션, 3D 뷰 회전, 감지 모델 설정 문서 추가
