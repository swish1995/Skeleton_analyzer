IMAS

버전 1.0.0

동영상 기반 자세 분석 및 인체공학 평가 도구

📋 프로그램 소개

IMAS는 작업 현장의 동영상을 분석하여 작업자의 자세를 평가하고, 근골격계 질환(MSD) 위험을 사전에 파악할 수 있도록 도와주는 전문 소프트웨어입니다.

최신 AI 기술인 MediaPipe Pose를 활용하여 동영상에서 실시간으로 인체의 33개 관절점을 추출하고, 이를 바탕으로 국제적으로 인정받는 5가지 인체공학 평가 방법을 자동으로 적용합니다.

✨ 주요 기능

RULA (Rapid Upper Limb Assessment)

상지(팔, 손목) 중심의 반복 작업에 적합한 평가 방법

점수 범위 1-7점
평가 부위 상박, 하박, 손목, 손목 비틀림, 목, 몸통, 다리

1단계: A그룹 점수 = TABLE_A[상박][하박][손목][손목비틀림]
2단계: B그룹 점수 = TABLE_B[목][몸통][다리]
3단계: 최종 점수 = TABLE_C[A그룹][B그룹]

OWAS (Ovako Working Posture Analysis System)

작업 자세를 4자리 코드로 분류하는 시스템

코드 구성 허리(1-4), 상지(1-3), 하지(1-7), 무게(1-3)
조치 등급 AC 1-4 (즉시 조치 필요 여부)

계산 공식

1단계: 자세 코드 = [허리][상지][하지][무게] (4자리)
2단계: 조치 카테고리 = ACTION_TABLE[허리, 상지, 하지]
예: "2121" = 굴곡 + 양팔아래 + 양다리서기 + 10kg미만

허리 (1자리)

코드분류기준
1직립몸통 굴곡 20° 이내
2굴곡몸통 굴곡 20° 초과
3회전/측굴어깨 또는 골반의 좌우 차이 감지
4굴곡+회전굴곡과 회전/측굴 동시 발생

상지 (2자리)

코드분류기준
1양팔 어깨 아래양쪽 손목/팔꿈치 모두 어깨 높이 아래
2한 팔 어깨 위한쪽 손목 또는 팔꿈치가 어깨 높이 이상
3양팔 어깨 위양쪽 손목 또는 팔꿈치 모두 어깨 높이 이상

하지 (3자리)

코드분류기준
1앉기수동 선택 (의자에 앉은 상태)
2양다리 서기양다리 곧음 (무릎 각도 150° 이상)
3한다리 서기한다리에 체중 실림
4양무릎 굴곡양무릎 굴곡 (무릎 각도 90°~150°)
5한무릎 굴곡한쪽 무릎만 굴곡
6무릎꿇기양무릎 각도 90° 미만
7걷기/이동움직이면서 작업

조치 카테고리 (위험 수준)

AC1: 정상 - 조치 불필요 AC2: 약간 유해 - 가까운 시일 내 개선 AC3: 명백히 유해 - 가능한 빨리 개선 AC4: 매우 유해 - 즉시 개선

NLE (NIOSH Lifting Equation)

들기 작업의 권장 무게 한계(RWL)를 산출

입력값 수평거리(H), 수직높이(V), 이동거리(D), 비틀림(A), 빈도(F), 손잡이(C)
결과값 RWL(kg), 리프팅 지수(LI)

계산 공식

RWL = LC x HM x VM x DM x AM x FM x CM
LI = 실제 중량(kg) / RWL
LC (Load Constant) = 23 kg

승수 요소 (Multipliers)

승수변수계산식설명
HMH (cm)25 / H (최대 1.0)수평 거리 (손-척추 중심선)
VMV (cm)1 - 0.003 x |V - 75|수직 높이 (바닥 기준)
DMD (cm)0.82 + 4.5 / D (최대 1.0)수직 이동 거리
AMA (도)1 - 0.0032 x |A|비대칭 각도 (몸통 회전)
FMF (회/분)표 참조빈도 (작업 시간 고려)
CMC (1-3)표 참조손잡이 품질

위험 수준 (LI 기준)

LI ≤ 1.0: 안전 1.0 < LI ≤ 3.0: 주의 LI > 3.0: 위험
계산 예시: H=40cm, V=75cm, D=50cm, A=30°, F=2회/분(1시간), C=1(Good), 실제무게=15kg
HM=0.625, VM=1.0, DM=0.91, AM=0.904, FM=0.91, CM=1.0
RWL = 23 x 0.625 x 1.0 x 0.91 x 0.904 x 0.91 x 1.0 = 10.77kg
LI = 15 / 10.77 = 1.39 (주의)

SI (Strain Index)

반복적인 손/손목 작업의 스트레인을 평가

입력값 강도, 지속시간, 빈도, 손목 자세, 작업 속도, 하루 지속시간
결과값 SI 점수 (3 미만: 안전, 7 이상: 위험)

계산 공식
