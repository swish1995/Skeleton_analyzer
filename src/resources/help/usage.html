🚀 시작하기

🎬 동영상 파일 열기

1
프로그램을 실행합니다
2
파일 → 동영상 파일 열기 메뉴를 선택하거나 Ctrl+O을 누릅니다
3
분석할 동영상 파일을 선택합니다 (MP4, AVI, MOV 등)

🖼️ 이미지 폴더 열기 (등록 기능)

1
파일 → 이미지 폴더 열기 메뉴를 선택하거나 플레이어의 이미지 폴더 버튼을 클릭합니다
2
이미지가 포함된 폴더를 선택합니다 (JPG, JPEG, PNG, BMP, TIFF 지원)
3
로딩 진행률이 표시되며, 완료 후 확인 버튼을 눌러 진행합니다
4
◀/▶ 버튼 또는 화살표 키로 이미지를 탐색하고 Enter로 캡처합니다
🔑 참고: 이미지 폴더 열기 기능은 라이센스 등록이 필요합니다.

📦 압축 파일 열기 (등록 기능)

1
파일 → 압축 파일 열기 메뉴를 선택하거나 플레이어의 압축 파일 버튼을 클릭합니다
2
ZIP 파일을 선택합니다
3
압축 해제 진행률이 표시되며, 완료 후 확인 버튼을 눌러 진행합니다
4
이미지 폴더와 동일한 방식으로 슬라이드쇼 탐색이 가능합니다
🔑 참고: 압축 파일 열기 기능은 라이센스 등록이 필요합니다.

📂 프로젝트 열기 (등록 기능)

1
파일 → 프로젝트 열기 또는 Ctrl+Shift+O
2
.skpx 확장자의 프로젝트 파일을 선택합니다
3
이전 작업 상태가 그대로 복원됩니다
🔑 참고: 프로젝트 저장/열기 기능은 라이센스 등록이 필요합니다.

1

발급받은 라이센스 키를 입력하고 등록 버튼을 클릭합니다
💡 팁: 라이센스 키는 하드웨어에 종속됩니다. 컴퓨터를 변경하면 새 키가 필요합니다.

🖥️ 화면 구성

프로그램 화면은 크게 좌측 영역과 우측 영역으로 나뉩니다.

◀️ 좌측: 비디오 플레이어

  • 동영상과 스켈레톤 오버레이 표시
  • 재생, 일시정지, 정지, 탐색 버튼
  • 타임라인 슬라이더
  • 캡처 버튼

▶️ 우측: 분석 패널

  • 상태 패널 - 현재 프레임의 관절 각도
  • 데이터 패널 - 캡처된 데이터 스프레드시트
  • 분석 결과 패널 - 움직임 빈도 분석 결과
  • 안전지표 패널 - 인체공학 평가 결과
💡 각 패널은 Ctrl+1, Ctrl+2, Ctrl+3, Ctrl+4 단축키로 표시/숨기기를 전환할 수 있습니다.

✏️ 스켈레톤 편집 🔑 등록 기능

관절 좌표를 직접 드래그하여 이동하면 RULA/REBA/OWAS 점수가 실시간으로 재계산됩니다. 영상 없이도 특정 자세의 점수를 확인하거나, 점수 로직을 테스트할 때 유용합니다.

🚀 편집 모드 시작하기

1
동영상을 로드하고 인체가 감지된 프레임에서 일시정지합니다
2
스켈레톤 시각화 패널에서 관절점을 클릭하면 자동으로 편집 모드에 진입하고 영상이 일시정지됩니다
3
관절을 드래그하여 이동하면 우측 각도/점수가 실시간 재계산됩니다
4
동영상을 다시 재생하면 편집 모드가 자동으로 해제됩니다

🖱️ 마우스 조작

동작 방법
관절 이동 (편집 자동 시작) 편집 가능 관절을 좌클릭 드래그
화면 이동 (팬) 편집 모드에서 빈 영역을 좌클릭 드래그
확대/축소 편집 모드에서 마우스 휠 또는 상단 +/- 버튼

🔧 컨트롤 바 버튼

버튼 설명
초기화 관절 위치를 편집 전 상태로 복원하고, 확대/축소/이동도 함께 초기화합니다
+ / - 확대 / 축소

🦴 편집 가능 관절

총 13개의 주요 관절을 편집할 수 있습니다:

상체

  • 코 (머리 방향)
  • 왼쪽/오른쪽 어깨
  • 왼쪽/오른쪽 팔꿈치
  • 왼쪽/오른쪽 손목

하체

  • 왼쪽/오른쪽 고관절
  • 왼쪽/오른쪽 무릎
  • 왼쪽/오른쪽 발목
💡 팁: 관절을 클릭하면 자동으로 편집 모드에 진입하고, 동영상을 다시 재생하면 자동으로 편집 모드가 해제됩니다. 초기화 버튼을 누르면 관절 위치와 확대/축소가 모두 원래 상태로 돌아갑니다.
🔑 참고: 스켈레톤 편집 기능은 라이센스 등록이 필요합니다.

📊 움직임 빈도 분석 🔑 등록 기능

동영상 전체를 순차 스캔하여 신체 부위별 움직임 횟수와 고위험 자세 노출 비율을 분석합니다. 실시간 평가(안전 지표)와 달리, 전체 영상에 대한 종합 통계를 제공합니다.

🚀 분석 시작하기

1
동영상을 로드합니다
2
분석 결과 탭을 클릭하거나 툴바의 분석 버튼을 클릭합니다
3
프레임 샘플링 옵션을 선택합니다 (전체 / 매 2프레임 / 매 3프레임, 기본: 매 2프레임)
4
분석 시작 버튼을 클릭합니다
5
분석이 완료되면 결과 테이블이 자동으로 표시됩니다

⏱️ 프레임 샘플링

긴 동영상의 경우 프레임 샘플링으로 분석 시간을 줄일 수 있습니다.

옵션 설명 분석 시간
전체 프레임 모든 프레임을 분석 (가장 정확) -
매 2프레임 (기본) 2프레임마다 1프레임 분석 ~50% 단축
매 3프레임 3프레임마다 1프레임 분석 ~67% 단축
💡 팁: 30분 이상의 동영상은 "매 2프레임" 또는 "매 3프레임"을 권장합니다. 샘플링 시에도 이전 분석 프레임 대비 각도 변화를 비교하므로 정확도가 크게 떨어지지 않습니다.

📋 결과 테이블 읽기

컬럼 설명
부위 13개 관절 부위 (목, 어깨, 팔꿈치, 손목, 엉덩이, 무릎, 발목)
움직임 인라인 바 + 횟수. 이전 프레임 대비 각도 변화가 15° 초과인 경우 카운트
고위험 비율 RULA/REBA 부위별 점수가 4점 이상인 프레임의 비율
평균 각도 분석된 프레임 전체의 평균 관절 각도

⚠️ 감지 실패 경고

포즈 감지 실패율이 30%를 초과하면 결과 상단에 경고 배너가 표시됩니다. 화질이 낮거나 사람이 가려진 구간이 많은 경우 발생할 수 있습니다.

⚠️ 참고: 감지 실패 프레임은 분석에서 제외되므로, 실패율이 높을수록 결과의 대표성이 낮아질 수 있습니다.

⏸️ 분석 취소 및 재개

  • 분석 중 취소하면 현재까지의 진행 상태가 저장됩니다
  • 취소 후 이어서 분석 버튼을 클릭하면 중단된 지점부터 다시 시작합니다
  • 재개 시 이전 분석 시간이 누적되어 정확한 총 소요 시간이 표시됩니다
  • 처음부터 시작 버튼을 클릭하면 저장된 진행 상태를 삭제하고 처음부터 다시 분석합니다

🔄 추가 안내

  • 새 동영상을 로드하면 이전 분석 결과가 자동으로 초기화됩니다
  • 분석이 시작되면 프레임 샘플링 옵션이 잠기며, 다시 분석 버튼으로 초기화해야 변경할 수 있습니다
  • 다시 분석 버튼을 누르면 결과가 초기화되고 프레임 옵션 선택 화면으로 돌아갑니다
  • 컬럼 헤더를 클릭하여 움직임 횟수순 또는 고위험 비율순으로 정렬할 수 있습니다
  • 분석은 재생 상태와 독립적으로 동작합니다 (재생 속도, 시크 등에 영향 없음)
  • 분석 결과는 프로젝트 저장 시 함께 저장되며, 프로젝트를 열면 복원됩니다

스프레드시트에서 우클릭하여 내보내기 메뉴에 접근합니다.

📄 JSON 내보내기

모든 캡처 데이터를 JSON 형식으로 저장합니다. 프로그래밍 방식의 데이터 처리에 적합합니다.

📊 Excel 내보내기

보고서 작성에 적합한 형식입니다. 이미지, 수식 포함 옵션을 선택할 수 있습니다.

⚙️ Excel 내보내기 옵션

옵션 설명
이미지 포함 비디오 프레임과 스켈레톤 이미지를 Excel에 포함합니다
이미지 크기 작게(50×50), 보통(150×150), 크게(200×200), 매우 크게(300×300)
행 높이 자동 이미지 크기에 맞춰 행 높이가 자동 조절됩니다
수식 포함 INDEX 함수를 사용한 수식이 포함되어 값 변경 시 자동 재계산됩니다
💡 팁: 수식 포함 시 숨겨진 시트(_RULA_Tables, _REBA_Tables 등)에 조회 테이블이 자동 생성됩니다.
